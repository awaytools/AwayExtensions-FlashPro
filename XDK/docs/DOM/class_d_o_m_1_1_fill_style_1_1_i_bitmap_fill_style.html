virtual FCM::Result _FCMCALL DOM::FillStyle::IBitmapFillStyle::IsClipped ( FCM::Boolean &  clipped)

This function can be used to check if the bitmap is larger than the fill area. If the bitmap is larger than the fill area, the bitmap is clipped; else it is repeated.

Parameters

Returns
On success, FCM_SUCCESS is returned, else an error code is returned.
