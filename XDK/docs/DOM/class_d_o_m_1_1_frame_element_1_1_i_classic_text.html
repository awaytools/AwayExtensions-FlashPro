virtual FCM::Result _FCMCALL DOM::FrameElement::IClassicText::GetParagraphs ( PIFCMList &  pParagraphList)
pure virtual

This function returns the list of paragraphs in the text element.

Parameters
pParagraphList(OUT) List of paragraphs in the text element. Each item in the list can be queried for IParagraph.
Returns
On success, FCM_SUCCESS is returned, else an error code is returned.
See also
IParagraph
Note
For empty dynamic and input text, this function will return a list with one paragraph in it which in turn will contain only one text run.
