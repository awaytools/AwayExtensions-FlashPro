virtual FCM::Result _FCMCALL DOM::FrameElement::ITextStyle::IsAutoKernEnabled ( FCM::Boolean &  autoKern)
pure virtual

This function returns true if kerning pair information in the font(s) to kern the text can be used. If it returns false, kerning pair information in the font(s) is ignored. It only works with static text and will be ignored if used with other text types.

Parameters
autoKern(OUT) Returns true if kerning pair info is to be used; else false is returned.
Returns
On success, FCM_SUCCESS is returned, else an error code is returned.
