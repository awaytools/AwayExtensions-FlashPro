pure virtual

This function is used to export a bitmap to a file.

Parameters
pBitmap(IN) An IMediaItem of a bitmap in libary.
pFilePath(IN) A string that specifies the absolute path (including name) of the exported file. The parent directory should exist. The file extension can be '.JPG' or '.PNG'.
quality(IN) A number, from 1-100, which determines the quality of the exported image file. A higher number indicates higher quality.
Returns
On success, FCM_SUCCESS is returned, else an error code is returned.
