Detailed Description

This structure contains placement information about a display object on the stage, that are not present in DISPLAY_OBJECT_INFO Given a DISPLAY_OBJECT_INFO pointer pDisplayObjectInfo, access the fileds in this structure only if the condition (pDisplayObjectInfo->structSize >= sizeof(DISPLAY_OBJECT_INFO_2)) is true.
