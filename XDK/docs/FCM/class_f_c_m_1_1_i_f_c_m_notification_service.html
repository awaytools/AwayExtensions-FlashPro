virtual FCM::Result _FCMCALL FCM::IFCMNotificationService::RegisterNotificationClient ( PIFCMNotificationClient  pClient,
ConstRefFCMNotifyID  nid,
const FCM::PVoid  pRefCon 
)
pure virtual

Registers a IFCMNotificationClient interface to receive notification whenever a specific type of event occurs. The type of event is specified by nid.

Parameters
pClient(IN) The client to be notified when the event occurs.
nid(IN) The id for the event for which the the client wants to be notified for.
pRefCon(IN) A refcon pointer that will be passed to the client at the time of notification.
Returns
FCM_SUCCESS is returned for success, else an error code is returned.
virtual FCM::Result _FCMCALL FCM::IFCMNotificationService::SendNotification ( ConstRefFCMNotifyID  nid,
const FCM::PVoid  pNotificationData 
)
pure virtual

Sends a notification call to all IFCMNotificationClients registered for the event specified by nid. Any data passed in the pNotificationData parameter will be passed on to the client interfaces in the pNotificationData parmaeter of the Notify method.

Parameters
pNotificationData(IN) This data will be passed on to all client interfaces.
nid(IN) The id for the event to be sent to all clients.
Returns
FCM_SUCCESS is returned for success, else an error code is returned.

ConstRefFCMNotifyID  nid

Unregisters an IFCMNotificationClient interface that was previously registered by a call to RegisterNotificationClient. The type of the event to unregister for is specified by nid.

nid(IN) The id of the event to unregister for.
Returns
