virtual FCM::Result _FCMCALL FCM::IFCMPluginDictionary::CreateRegistryEntry ( ConstRefFCMCLSID  classID,
ConstRefFCMIID  interfaceID,
PIFCMDictionary *  ppRegistryDict 
)
pure virtual

Create a registry dictionary and adds it to the list of registry dictionaries. Optionally return the new dict (if ppRegistryDict is not NULL).The registry dict returned by CreateRegistryEntry is owned by the cache, and you MUST NOT call Release() on it (unless you also call AddRef() on it).

Parameters
classID(IN) The class Id of the registry dict.
interfaceID(IN) The interface Id of the registry dict.
ppRegistryDict(OUT) Set with the new dictionary created.
Returns
FCM_SUCCESS is returned for success, else an error code is returned.
